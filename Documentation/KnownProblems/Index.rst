Known problems
--------------

If you have any issues, please post about your problems to the TYPO3
English mailing list (typo3.english), so that others may benefit from
the answers too. You may also refer to the Tesseract Project web site
(http://www.typo3-tesseract.com/).

For bugs or feature requests, please open an entry in the extension's
bug tracker on Forge (http://forge.typo3.org/projects/extension-context/issues).
